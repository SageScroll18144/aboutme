PROJETOS

chess-ai

Projeto tem como objetivo desenvolver uma inteligência artificial baseada em algoritmos clássicos de teoria dos jogos para o jogo Xadrez! O algoritmo é baseado numa árvore que cada nó corresponde à uma possivel futura situação do tabuleiro e a decisão é feita utilizando Alpha-Beta Pruning, uma otimização do Min-Max. | Java.

Eye Tracking

Projeto tem como objetivo o rastreio dos olhos como base algoritmos clássicos de visão computacional. | Python.
